# 🚴 Pulini Cycles

Site oficial da **Pulini Cycles** — loja de bicicletas, Service Center técnico e organizadora de viagens e passeios ciclísticos.

## Sobre o projeto

Site one-page desenvolvido em HTML5 + CSS puro, com foco em conversão para WhatsApp e captura de leads.

## Seções

- **Hero** — Apresentação e CTA principal
- **Serviços** — Revisão Geral, Suspensões Fox/RockShox, Eletrônica Di2/AXS
- **Viagens** — Caminho da Fé, Santiago de Compostela
- **Passeios** — Grupos semanais
- **Comunidade** — Strava e Instagram
- **Sobre** — Loja Giant oficial
- **Contato** — Formulário de leads integrado ao Supabase

## Stack

- HTML5 + CSS3 puro (sem frameworks)
- Google Fonts — Inter
- Integração Supabase para captura de leads
- Deploy via Vercel
